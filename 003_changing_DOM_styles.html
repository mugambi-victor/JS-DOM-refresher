DOM manipulation using JavaScript: Styles

There are two main ways of styling elements when working with JavaScript. You can either use the .style property or you can use classes

.style

You use the .style property when you want to make specific changes to a single element. The .style property allows you to set styles directly as inline styles for elements. This means that it overrides the styles you have in your CSS stylesheet

The classList Property

To add a new class instead of replacing the existing class, we use the classList property. With the classList property, you can add and remove classes. You can also toggle, replace existing class values with new ones, and even check if a class contains a specific value.

Example:

I like rice.

Useful method

  • classList.add() adds classes to an element
  • classList.remove() removes classes from an element
  • classList.toggle() adds or removes a class from an element
  • classList.contains() checks if an element has a specific class. returns true or false
  • classList.replace() replaces an existing class with a new one
