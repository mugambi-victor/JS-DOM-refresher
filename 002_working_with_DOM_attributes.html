Html Attributes provide useful information about html elements. These Attributes are always included in the opening tag of the element. An Attribute is made up of a name and a value.

As the browser generates the DOM based in the HTML structure, it translates the attributes into dynamic properties of the DOM objects.

Example

To interact with attributes, use the method getAttrribute and setAttribute to directly access the properties.

The getAttribute Method

  • As the name suggests, you can use this method to get the value of an existing attribute on an element
  • It accepts an argument(the name of the attribute) and returns the value of the attribute.

The setAttribute Method

This is used to set or update the attribute of an element. The method taked in two arguments. The first argument is the name of the attribute and the second attribute is the new value you want to give the attribute for example. in our script we will update the src attribute of out image to a randdom image generator instead of the current image.

The removeattribute Method

Used to remove an existing attribute. eg we can remove the height attribute of our image

The hasAttribute Method

This is another useful method when manipulating the Dom to check if an element has a given attribute. The method returs true if the specified attribute exists and returns false
